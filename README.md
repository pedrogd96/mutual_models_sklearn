# Projeto de Machine Learning

Este projeto contém a implementação e análise de diferentes modelos de **Machine Learning supervisionado**, com foco na avaliação de desempenho e interpretação dos resultados.

## Instalação

Após clonar o repositório, instale todas as bibliotecas necessárias executando o comando abaixo no diretório raiz do projeto:

```bash
pip install -r requirements.txt
```

## Execução

Para rodar uma classe Python e visualizar seus resultados diretamente pelo **CMD / terminal**, utilize o comando abaixo:

```bash
python nome_do_arquivo.py
```

Exemplo:

```bash
python Conclusion/ConclusionModel.py
```

Certifique-se de executar o comando a partir da **raiz do projeto**.

## Análises dos Modelos

As análises completas dos modelos de Machine Learning utilizados neste projeto estão documentadas no arquivo PDF disponível na raiz do repositório:

## Fundamentos de Machine Learning com Scikit-Learn - Pedro Duarte.pdf

O documento inclui:

* Descrição do dataset utilizado
* Metodologia de treinamento
* Comparação entre os modelos
* Métricas de avaliação
* Interpretação dos resultados
